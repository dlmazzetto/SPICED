Exploratory vs. Explanatory Data Visualization¶

Exploratory Data Visualization¶

Chart types¶

  • From Data to Viz: Gallery of many possibilities of graphs to use: from Data to Viz leads you to the most appropriate graph for your data. It links to the code to build it and lists common caveats you should avoid.
  • Python Graph Gallery: This website displays hundreds of charts, always providing the reproducible python code.

Good exploratory visualization habits¶

  • Think about data and chart type you need to visualize the data relationships to answer your questions
  • Visualize data quality e.g. missing values (Lesson: Data Wrangling)
  • Take a look at your variables e.g. distribution shape, correlation (Lesson: Descriptive statistics)

Some rules for effective explanatory data visualization¶

  • Design for your AUDIENCE: form follows function. Question and modify defaults.
  • Get rid of chart junk: elements than do not contribute to clarifying the message.
  • Use color with intention: https://academy.datawrapper.de/article/140-what-to-consider-when-choosing-colors-for-data-visualization
